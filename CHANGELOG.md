# Change Log

All notable changes to the "macOS mi Style" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.0.7] - 2026-01-08

### Added
- Laravel Blade ディレクティブ (`@if`, `@foreach`, `@extends`, `@section` など) のシンタックスハイライト対応
- Livewire属性 (`wire:model`, `wire:click`, `wire:submit` など) のカラーリング最適化
- Blade内のPHP文字列とメソッド/関数呼び出しの色分け改善
- Alpine.js属性 (`x-data`, `x-show`, `x-on` など) のハイライト対応

### Changed
- Laravel開発に最適化されたカラースキームに更新
- Blade テンプレート内での視認性を向上
